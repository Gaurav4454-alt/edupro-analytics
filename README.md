## EduPro Analytics Dashboard

## Project Overview
EduPro Analytics Dashboard is a data analytics project designed to analyze instructor performance, course quality, ratings, enrollments, and business insights.

## Features
- KPI Dashboard
- Instructor Leaderboard
- Category Performance Analysis
- Correlation Analysis
- Interactive Streamlit Dashboard
- Data Visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Plotly
- Streamlit
- Excel

## Project Structure
- EduPro_Analytics_Report.xlsx
- EduPro_Research_Paper.md
- app.py
- requirements.txt
- README.md
- EduPro_Online_Platform.xlsx
